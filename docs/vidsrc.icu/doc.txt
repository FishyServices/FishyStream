API Documentation
Detailed representation of the API endpoints for Vidsrc includes comprehensive information regarding the available methods, request formats, required parameters and optional parameters.
Endpoint
Valid parameters:

- {id} required - from imdb.com or themoviedb.com, imdb id must have tt prefix.
Examples

https://vidsrc.icu/embed/movie/tt17048514

https://vidsrc.icu/embed/movie/927085
Endpoint
Valid parameters:

- {id} required - from imdb.com or themoviedb.com, imdb id must have tt prefix.

- {season} optional - the eason number

- {episode} optional - the episode number
Examples

https://vidsrc.icu/embed/tv/tt18382028/1/5

https://vidsrc.icu/embed/tv/158876/1/5

https://vidsrc.icu/embed/tv/927085

Endpoint
Valid parameters:

- {id} required - from https://anilist.co, anilist id needed

- {episode} required - the episode number

- {dub} optional - Put 0 for sub and 1 for dub
Examples

https://vidsrc.icu/embed/anime/21/1/0 - sub

https://vidsrc.icu/embed/anime/21/1/1 - dub
Endpoint
Valid parameters:

- {id} required - from https://anilist.co, anilist id needed

- {chapter} required - the chapter number
Examples

https://vidsrc.icu/embed/manga/30013/1
